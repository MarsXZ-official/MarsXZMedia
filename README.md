MarsXZ Media — это кроссплатформенный инструмент (Windows и Android) для загрузки видео и аудио с YouTube. Программа предназначена для быстрого и удобного скачивания медиафайлов.

В зависимости от платформы, приложение использует разные внутренние компоненты и инструменты.

## 🎥 Поддерживаемые платформы (Источники)

В текущей версии поддерживается загрузка только с **YouTube**.  
Поддержка других платформ планируется в будущих версиях.

Разрешённые форматы ссылок:
- `https://youtu.be/...`
- `https://www.youtube.com/...`
- `https://m.youtube.com/...`

## 💻 Версия для Windows

Настольное приложение работает как самостоятельный исполняемый файл и оптимизировано для слабых ПК. **Программа работает без установки .NET.**

### Технологический стек (ПК)
- **yt-dlp** — загрузка медиа с YouTube
- **ffmpeg** — обработка, объединение и конвертация файлов (MP4/MP3)
- **node** — работа внутренних компонентов приложения

*(Встроена система восстановления файлов: если нужные компоненты отсутствуют, программа загрузит их автоматически.)*

### Системные требования (ПК)
- **Windows / Legacy**: Windows 8 и выше
- **Windows / Modern**: Windows 10 (версия 18362 и выше)
- **Архитектура**: x86 / x64
- **Свободное место**: ~300 МБ
- Интернет-соединение

## 📱 Версия для Android

Android-версия переписана с нуля для мобильных устройств и адаптирована под сенсорный интерфейс.

На текущем этапе приложение сосредоточено на основной функции загрузки и будет постепенно развиваться в следующих версиях.

### Технологический стек (Мобильная версия)
- **Kotlin & Jetpack**
- **NewPipeExtractor** — получение информации о медиа и доступных потоках
- **Custom Downloader** — внутренняя логика загрузки (`DownloaderImpl`)
- **FFmpeg Mobile** — склейка адаптивных потоков видео и аудио, а также конвертация медиафайлов

### Системные требования (Мобильная версия)
- **ОС**: Android 5.0 (API 21) и выше
- **Архитектура**: arm64-v8a, armeabi-v7a, x86_64, x86
- **Свободное место**: ~150 МБ
- Интернет-соединение

## 📥 Установка

### Для Windows
1. Перейдите в раздел [Releases](https://github.com/MarsXz8656/MarsXZMedia/releases/latest)
2. Загрузите установочный файл `.exe`
3. Следуйте инструкциям мастера установки
4. Запустите MarsXZ Media

### Для Android
1. Перейдите в раздел [Releases](https://github.com/MarsXz8656/MarsXZMedia/releases/latest)
2. Скачайте актуальный файл `.apk`
3. Разрешите установку из неизвестных источников в настройках смартфона
4. Установите и запустите приложение
